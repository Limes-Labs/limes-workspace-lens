# Project Rules

- Public docs are written in English.
- Synthetic fixtures are never presented as model evidence.
- Readout hits are hypothesis-generation signals until paired with behavior and controls.
- External code is not vendored without an `UPSTREAMS.md` update and license review.
- Real model runs must record checkpoint, tokenizer, lens, prompts, commands, and compute context.
- Negative and mixed results stay in the repo when they inform the boundary of the method.
